INSTALLATION
------------
1. Double-click  install.bat
2. Follow the on-screen steps
3. Click Y to launch when asked

RUNNING THE APP
---------------
After install, use any of these:
  • Desktop shortcut  → "AI Java Interview"
  • Start Menu        → search "AI Java Interview"
  • Browser URL       → http://localhost:8501/JavaAIMockInterview/
  • Manually          → double-click run_app.bat
  • Browser           → http://localhost:8501

FEATURES
--------
  ☕ Java Mock Interview
     - 10 topic areas (Core Java, Spring Boot, DSA, etc.)
     - 3 difficulty levels (Junior / Mid / Senior)
     - 3–15 questions per session
     - 2–5 minute countdown timer per question
     - AI scores every answer 0–10 with detailed feedback
     - Voice input (speak your answer) + Voice output (AI reads questions)
     - Auto-advance when time expires
     - Final Report Card with AI overall analysis

  🌍 Multilingual Assistant
     - English ↔ Hindi Tutor
     - English ↔ Spanish Tutor
     - System Assistant
     - Customer Support (Groq APIs)
     - Log/Text file analyser

REQUIREMENTS
------------
  - Windows 10 / 11
  - Python 3.9 or higher
  - Internet connection (for Groq API + Google Speech Recognition)
  - Microphone (for voice input)
  - Speakers / headphones (for voice output)

TROUBLESHOOTING
---------------
  Q: App doesn't open?
  A: Run run_app.bat manually and check the CMD window for errors.

  Q: Voice not working?
  A: Make sure your microphone is set as the default recording device.
     Check Windows Settings → Sound → Input.

  Q: "No module named streamlit" error?
  A: Re-run install.bat to reinstall dependencies.

  Q: API error / no response?
  A: Check your internet connection. The Groq API key is embedded in the app.
